# Ahmed's Smart School Schedule 🎓

A personalized, intelligent school bell schedule developed by **Ahmed Assaidy** for the 2025-2026 academic year at **Origins High School**.

## ✨ Features
* **Automatic Day Detection:** The system intelligently switches between the **Regular Day** and **Wednesday** schedules based on your device's date.
* **Real-time Countdown:** A high-precision timer showing exactly how much time is left in the current class or break.
* **Live Status Updates:** Each period automatically updates its status to **Active**, **Ended**, or **Wait** as the day progresses.
* **Dynamic Progress Bar:** A visual blue-themed bar that fills up in real-time during your classes.
* **Minimalist Blue Theme:** A sleek, modern design with a dark-blue aesthetic and neon glow effects.

## 🚀 How it Works
No manual switching is required. Just open the site, and it will:
1.  Check the current day of the week.
2.  Load the correct bell times (57-min periods for regular days, 46-min for Wednesdays).
3.  Highlight your current class and start the countdown.

## 🛠 Tech Stack
* **HTML5:** Structured for clarity and accessibility.
* **CSS3:** Custom styles using CSS variables and glassmorphism.
* **JavaScript (ES6+):** Logic for time calculations and automatic schedule switching.
